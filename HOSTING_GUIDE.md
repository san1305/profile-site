# Sanu Dutta — Profile Website

A 5-page interactive personal website built as static HTML/CSS/JS. No build step, no dependencies — it runs anywhere.

## Pages
- `index.html` — Home (hero, animated stats, signature deals)
- `expertise.html` — Interactive capability tabs + skill bars
- `portfolio.html` — Filterable deal cards
- `experience.html` — Career timeline, education, certifications
- `contact.html` — Contact details + message form (opens email)
- `assets/styles.css`, `assets/main.js` — shared design + interactivity

## Preview locally
Just double-click `index.html` — it opens in your browser. That's it.

## Hosting options (free, ranked easiest first)

### 1. Netlify Drop — fastest, no account setup needed
1. Go to https://app.netlify.com/drop
2. Drag the whole `Profile_Website` folder onto the page.
3. You get a live URL in ~10 seconds (e.g. `sanu-dutta.netlify.app`).
4. Free custom domain support if you buy one later.

### 2. GitHub Pages — best if you want a permanent, editable home
1. Create a free GitHub account.
2. Make a new repository named `sanudutta.github.io`.
3. Upload all files from this folder (keep the `assets/` folder structure).
4. Site goes live at `https://sanudutta.github.io`.

### 3. Cloudflare Pages / Vercel
Similar to Netlify — connect a GitHub repo or drag-drop. Good if you want analytics.

## Custom domain (optional)
Buy something like `sanudutta.com` (~USD 10/yr from Namecheap/GoDaddy) and point it at
whichever host you chose. All three hosts above give free HTTPS and simple domain setup.

## To update content later
Everything is plain text — open any `.html` file, edit the words, save. No coding tools needed.

---
*Recommendation: Start with **Netlify Drop** to get it live today, then move to GitHub Pages once you want a custom domain.*
